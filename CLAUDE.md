# Project Context
The built website must be as close to vanilla HTML5, CSS, and JS possible. End users should not be aware, as much as is possible, of the underlying md file datastore.

## About
This is my personal website. It is written on SvelteKit. It is inteded to be a minimal presentation focused on information about events and musical projects. The final website must be performant, using prerendering wherever possible. 

## key Directories
- '/src/lib' - stores all page/post info
- '/src/assets/images' - all images used with SvelteKit's enhanced image functionality
- '/src/routes' - all site routes
- '.idea' - all new files that you create MUST be created in this directory

## Standards
- Commented TypeScript
- Follow all accessibility best practices
- SEO/GEO aimed at surfacing Jonathan Piper when users try to learn about tubists/experimental musicians in San Diego
